# Timetable App

This is an improved version of the Timetable App, built with React and Tailwind CSS.

## Features
- **Live Dashboard:** Shows current class and countdown.
- **Attendance Tracking:** Click on a class to mark attendance. Data is saved automatically.
- **Persistence:** Attendance data survives page reloads (saved to LocalStorage).
- **Faculty Directory:** View staff details.
- **Stats:** View attendance summaries.

## How to Run
1. Open a terminal in this folder.
2. Run `npm install` (if not already done).
3. Run `npm run dev`.
4. Open the link shown (usually `http://localhost:5173`).

## How to Install on Mobile
1. Run with `npm run dev -- --host`.
2. Open the Network URL on your phone (must be on same WiFi).
3. Use Chrome/Safari "Add to Home Screen" to install it as an app.
